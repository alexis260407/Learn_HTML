Les bases:

I: Le texte

Pour créer une texte, il existe plusieurs manière différentes d'en créer.
Tout comme le <p> </p>
<h0> </h0> (de 1 jusqu'a 3).

Mais, vous pouvez en créer aussi, en intégrant un code style css, vous pouvez en créer un rien que pour vous.
Mais Attention, il faut tout de fois le config afin d'adapter à votre choix d'écriture, de taille ou même de largeur.

Par exemple ici, j'ai crée un h5 qui a une taille, la taille de la police et une couleur que je voulais avoir.

Maintenant, il faut savoir config le CSS.

Pour configurer le css, il faut mettre une lettre, un mot, ce que vous voulez dans cette case :

### il est pas conseillé de le faire, mais je le fais pour mieux comprendre mon code

Puis après nous pouvons implémenter:
La couleur:
color: (nom de la couleur en anglais ou en #couleur numérique);

Le Texte/Police:
 font-family: (nom de la police qu'on veut utiliser);
font-weight: (la hauteur du texte, mettre en "normal");
font-size: (permet de modifier la taille du texte);
text-align: (permet d'aligner quelque part le texte) ici (mettre un lien);
